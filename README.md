# FlowDesk AI

AI-Powered WhatsApp Business OS — A modern admin dashboard for managing customer interactions, analytics, payments, automation, and AI-assisted workflows.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Charts:** Recharts

## Features

- **Dashboard Overview** — Real-time metrics, revenue charts, and engagement summaries
- **Customer Management** — Contact list, interaction history, and segmentation
- **Inbox** — Unified messaging interface for customer conversations
- **Analytics** — Detailed reports on messaging volumes, response times, and growth
- **Automation** — Configure automated workflows and chatbot responses
- **Payments** — Transaction history, invoice management, and payout tracking
- **AI Assistant** — AI-powered reply suggestions and conversation insights
- **Settings** — Account configuration, notification preferences, and team management

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and click **Sign in** (demo credentials are pre-filled).

## Build

```bash
npm run build
npm start
```
